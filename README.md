# Sarier Inc.

Sitio web de Sarier Inc. — casa de marca para moda.
Sistema operativo para marcas de moda: estrategia, pauta, contenido, datos y desarrollo.

## Divisiones
- **Fashionalytics** — inteligencia y predicción de demanda
- **Studio** — operación creativa y de medios
- **Sarier Systems** — desarrollo y transformación digital

## Deploy
Sitio estático. Se publica automáticamente en Netlify con cada push a `main`.
Publish directory: raíz del repo. Archivo principal: `index.html`.
